# Features
* UAV制御機能
* RSSI計測機能
* 画像処理機能
* 部屋のレイアウト取得昨日
* ガス濃度取得機能
* 管理サーバでのデータ集約機能
* webページ表示機能

# Requirement

必要なライブラリ

* OpenCV(画像処理)
* noble(nodeでのble取得)
* node-bebop(UAVの制御)
